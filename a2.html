Circuit

For the circuit, I used Blue, Red, RGB and Yellow LEDs. I used a resistance of 220 Ohm for the Red, Yellow LEDs and the Red and Green prongs of the RGB LED because after doing the calculations, I found that the maximum Resistance I would need is 160 ohms. 220 was the closest standard resistance value. For the Blue LED I found that the maximum resistance I would need for the Blue LED is 85 Ohm. I used 100 Ohm for it because it was the closst standard to that resistance value.


Schematic and Resistance

Red, Yellow and Green all needed a minimum of 160 Ohm. Blue needs a minimum of 85 Ohm. The button needs a minimum of 250 Ohm, and while there is a close standard, we use a 10k Ohm resistor to limit the current to be 0.5 mA just to be really really safe and draw less power so that the Arduino does not blow up. It is also standard to use a 10k resistor for buttons.

The resistors used for each Red, Yellow LEDs and Red, Green prongs are 220 Ohm. The exact resistance needed was 160 Ohm. The resistors used for the Blue LED are 100 Ohm.

            const int red = 6;
            const int green = 9;
            const int blue = 11;
            
            void setup() {
              // put your setup code here, to run once:
              pinMode(red, OUTPUT);
              pinMode(green, OUTPUT);
              pinMode(blue, OUTPUT);
              pinMode(13, OUTPUT);
              pinMode(2, OUTPUT);
              pinMode(4, OUTPUT);
              pinMode(12, INPUT);
            }
            
            void loop() {
            
              while (digitalRead(12) == HIGH) {
            
                digitalWrite(13, HIGH);
                digitalWrite(4, HIGH);
                digitalWrite(2, HIGH);
            
                // Fade in Red and fade out Green
                for (int i = 0; i <= 255; i++) {
                  analogWrite(red, i);       // Increase Red brightness
                  analogWrite(green, 255 - i); // Decrease Green brightness
                  delay(10); // Small delay for smooth fading
                }
            
                // Fade in Blue and fade out Red
                for (int i = 0; i <= 255; i++) {
                  analogWrite(blue, i);      // Increase Blue brightness
                  analogWrite(red, 255 - i); // Decrease Red brightness
                  delay(10); // Small delay for smooth fading
                }
            
                // Fade in Green and fade out Blue
                for (int i = 0; i <= 255; i++) {
                  analogWrite(green, i);     // Increase Green brightness
                  analogWrite(blue, 255 - i); // Decrease Blue brightness
                  delay(10); // Small delay for smooth fading
                }
            
                digitalWrite(13, LOW);
                digitalWrite(4, LOW);
                digitalWrite(2, LOW);
              }
            }
